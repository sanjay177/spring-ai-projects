# Basic Ollama Connect

This module demonstrates how to integrate and use the Spring AI framework with the Ollama model for chat-based applications.

## Features

- Spring Boot application
- Integration with Spring AI and Ollama model
- REST API for chat functionality

## Requirements

- Java 21
- Maven 3.8+
- Spring Boot 3.1.4
- Ollama model support
